# Instrucciones para PWA - VIGIA PLOT

## Iconos Requeridos

Para que la PWA funcione correctamente, necesitas crear dos iconos:

1. **icon-192.png** - 192x192 píxeles
2. **icon-512.png** - 512x512 píxeles

Coloca estos archivos en la carpeta `public/`

## Crear los Iconos

Puedes usar cualquier herramienta de diseño para crear los iconos. El diseño debe incluir:
- Logo o símbolo de VIGIA PLOT
- Colores: Azul (#6496ff) y negro (#0a0e27)
- Fondo transparente o sólido oscuro

## Instalación en Tablet

1. Abre la aplicación en Chrome/Edge en la tablet
2. Verás un banner o menú con opción "Instalar" o "Agregar a pantalla de inicio"
3. Haz clic en instalar
4. La app se instalará como aplicación nativa
5. Se abrirá en modo standalone (sin barra del navegador)

## Verificación

- La app debe abrirse en pantalla completa
- Debe funcionar offline (con Service Worker)
- Debe tener icono en la pantalla de inicio
- Debe tener nombre "VIGIA PLOT"
